# Swiggy Sales Analysis Dashboard | Excel

## Project Overview
Analyzed Swiggy food delivery sales data to identify revenue trends, 
regional performance, and restaurant category insights using Excel.

## Dataset
- Source: Kaggle
- Records: ~1.97L orders across multiple Indian cities and states
- Period: January to August

## Key Insights
- Bengaluru alone generates ₹54.5L in sales — nearly equal to the 
  combined revenue of the next 4 top cities
- Q3 sales dropped ~32% compared to Q1/Q2, indicating possible 
  seasonality or data gap worth investigating
- Average rating is identical (4.34) across all quarters — suggests 
  rating system may not be a reliable performance differentiator

## Dashboard Features
- KPIs: Total Sales, Total Orders, Avg Order Value, Avg Rating
- Slicers: Month, State, Food Type, Category
- Charts: Monthly trend, Daily sales pattern, Regional map, 
  Top 10 restaurants, Top 5 cities, Food type split

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers, Conditional Formatting)
